# @armand-veress/string-toolkit

A small, useful JavaScript library for password utility functions.

## Context
This project was developed as part of a Continuous Integration course exercise. 

## Installation

Install using npm:
```bash
npm install @armand-veress/password-toolkit
```

Or using yarn:
```bash
yarn add @armand-veress/password-toolkit
```

## Development
To run tests:
```bash
npm test
```
